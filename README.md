## ✨ Features

- ✅ **Responsive Design** — Mobile-first, works on all devices
- ✅ **Glass Blur Navbar** — Frosted glass with backdrop-filter
- ✅ **Theme Toggle** — Dark/Light mode with localStorage
- ✅ **RTL Toggle** — Right-to-left language support
- ✅ **Animated Gradient Background** — CSS-only animated gradients
- ✅ **Scroll Animations** — AOS.js for reveal-on-scroll effects
- ✅ **Swiper Carousel** — Experience page highlight reels
- ✅ **Dynamic Tour List** — Shows rendered from JavaScript array
- ✅ **Dynamic Merch Grid** — Shop items rendered from JavaScript array
- ✅ **Booking Inquiry Form** — Full contact form with validation
- ✅ **Newsletter Signup** — Email subscription on every page
- ✅ **Consistent Layout** — Same navbar, footer, and design across all pages
- ✅ **Two Homepage Designs** — Cinematic (V1) and Structured (V2)

## 🚀 Deployment

This project is designed for **GitHub Pages** deployment:

1. Push to a GitHub repository
2. Enable GitHub Pages in Settings → Pages → Source: main branch
3. All paths are relative — works out of the box
